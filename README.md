# FluidSIM-Time-Changer
A simple time changer for the 30 Minutes limited Demo version of FluidSIM.

This Python script extends the 30-minute session limit of the FluidSIM Demo by automating system time manipulation.

Requirements:
- Windows 10/11
- Admin Rights
- Windows Time Service must be enabled 

Installation:
1] Download the FluidSIM Time Changer.py File
2] Open the File
3] Enter the path to your FluidSIM.exe File
4] Enter your desired time for FluidSIM to work, without closing.

Extra:
If you need help finding your path to the FluidSIM.exe File look into the .pdf
