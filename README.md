# FluidSIM-Time-Changer
A simple time changer for the 30 Minutes limited Demo version of FluidSIM.


Download Link: https://github.com/LaptopCpu/FluidSIM-Time-Changer/archive/refs/heads/main.zip


Requirements:
- Windows 10/11
- Admin Rights
- Windows Time Service must be enabled
- Python 3.x



Installation:
- Download the FluidSIM Time Changer.py File
- Open the File
- Enter the path to your FluidSIM.exe File
- Enter your desired time for FluidSIM to work, without closing.



Extra/Warning:
- Whilst running the FluidSIM Time Changer some websites might not load due to your system time being changed! 
- If you need help finding your path to the FluidSIM.exe File look into the .pdf
